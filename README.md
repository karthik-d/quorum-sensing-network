# Quorum Sensing and Network Formation

Agent-based quorum sensing model to analyze the coordination of macrophages in their trimodal pro-inflammatory response.

- Mathematica notebooks of the quorum model.
- Object-oriented implementation in Python.
- Python scripts for downstream analysis.

## Analysis Scripts

- Prefixed `B*_` are scripts used for baseline analysis—a single timepoint across different densities, seeding inits, etc.
- Prefixed `T*_` are scripts used for temporal analysis—comparing different timepoints of a given experiment / model setup.
